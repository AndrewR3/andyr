Git, GitHub

Andy Robinson May 2019

In programming, typically a group of developers will share code to complete a project, and split up the work. They do this using a version control system (VCS). The VCS we're using/learning is GitHub, not to be confused with Git. Git is the program used on the computer, GitHub is the software that stores the data on the cloud.
